Аналіз ринку крафтового пива в Україні



Мета: Дослідити стан, географію та динаміку розвитку малих виробників пива (до 3000 гектолітрів на рік) на основі відкритих даних України.



Джерело даних: Ліцензійний реєстр виданих ліцензій на оптову торгівлю алкогольними напоями, виключно пивом

(https://data.gov.ua/dataset/5c19fe03-fe99-457b-9051-8d280e68172b/resource/bc42ab1f-6f38-4c15-b16d-c475c6aa08bd)



Гіпотези для аналізу:



1. Більшість броварень реєструються як ТОВ (Юридичні особи), а не як ФОП.

2\. Існує сезонність: найбільша кількість ліцензій видається навесні, перед початком літнього сезону.

3\. Понад 15% виданих ліцензій наразі є анульованими, що вказує на складнощі ведення крафтового бізнесу.
